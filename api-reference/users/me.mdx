---
title: Get Current User
api: GET https://api.teel.finance/api/users/me
description: Get the currently authenticated user's profile
---

Returns the profile of the currently authenticated user based on their JWT token.

## Authentication

This endpoint requires a valid JWT bearer token in the `Authorization` header.

## Response

<ResponseField name="id" type="string">
  Unique user identifier.
</ResponseField>

<ResponseField name="email" type="string">
  User's email address.
</ResponseField>

<ResponseField name="name" type="string">
  User's display name.
</ResponseField>

<ResponseField name="auth0Id" type="string">
  The user's Auth0 identifier.
</ResponseField>

<ResponseField name="businessId" type="string">
  The ID of the business associated with this user, if any.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the user was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of the last profile update.
</ResponseField>
