---
title: Get Onboarding Status
api: GET https://dev.teelapp.io/api/kyb/onboarding/status
description: Get the KYB onboarding status across all providers
---

Returns the current onboarding status for the authenticated user's business across all connected providers. Requires a valid JWT.

## Authentication

This endpoint requires a valid JWT bearer token in the `Authorization` header.

## Response

<ResponseField name="overallStatus" type="string">
  Aggregate onboarding status across all providers (e.g. `pending`, `partial`,
  `complete`).
</ResponseField>

<ResponseField name="providers" type="object">
  Map of provider names to their individual onboarding status.
  <Expandable title="Provider status object">
    <ResponseField name="status" type="string">
      Current status for this provider (e.g. `pending`, `approved`, `rejected`, `not_started`).
    </ResponseField>

    <ResponseField name="updatedAt" type="string">
      ISO 8601 timestamp of the last status update.
    </ResponseField>

    <ResponseField name="reason" type="string">
      Reason for rejection, if applicable.
    </ResponseField>

  </Expandable>
</ResponseField>
