title: "Quickstart"
description: "Get up and running with the Teel API in 5 minutes"
---

This guide walks you through making your first cross-border payout using the Teel API.

<Steps>
  <Step title="Get Your API Key">
    Create an API key from the **API Keys** page in your [Teel dashboard](https://app.teel.finance). New keys require activation — contact **dahui@teelapp.io** with your use case.

    Once activated, set it as an environment variable:

    ```bash
    export TEEL_API_KEY="teel_live_your_api_key_here"
    ```

  </Step>

  <Step title="Check Coverage">
    Verify which currencies and corridors are available for your account based on your KYB status.

    ```bash
    curl -X GET https://dev.teelapp.io/api/kyb/capabilities/currencies \
      -H "X-API-Key: $TEEL_API_KEY"
    ```

    The response lists all currencies you can send and receive. Ensure your desired corridor (e.g., USD to PHP) is supported before proceeding.

  </Step>

  <Step title="Create a Recipient">
    Register the person or business you want to send funds to.

    ```bash
    curl -X POST https://dev.teelapp.io/api/recipients \
      -H "X-API-Key: $TEEL_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "business_name": "Acme Corp Philippines",
        "email": "payments@acme-ph.com",
        "country": "PH",
        "bank_details": {
          "account_name": "Acme Corp Philippines",
          "account_number": "1234567890",
          "bank_code": "BDO"
        }
      }'
    ```

    Save the `id` from the response. This is your `recipient_id`.

  </Step>

  <Step title="Get a Quote">
    Fetch the best available rate across all providers for your corridor.

    ```bash
    curl -X GET "https://dev.teelapp.io/api/quotes/fiat-to-fiat/best?source_currency=USD&target_currency=PHP&amount=1000" \
      -H "X-API-Key: $TEEL_API_KEY"
    ```

    The response includes the exchange rate, fees, estimated delivery amount, provider selected, and the quote TTL (expiry). You must use the quote before it expires.

  </Step>

  <Step title="Create a Payout">
    Execute the payout using the quote and recipient from the previous steps.

    ```bash
    curl -X POST https://dev.teelapp.io/api/payouts \
      -H "X-API-Key: $TEEL_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "quote_id": "quote_abc123",
        "recipient_id": "recipient_xyz789",
        "source_currency": "USD",
        "target_currency": "PHP",
        "amount": 1000,
        "reference": "invoice-2024-001"
      }'
    ```

    The payout is created in `pending` status and begins processing immediately.

  </Step>

  <Step title="Track Status">
    You can check payout status at any time via the REST API:

    ```bash
    curl -X GET https://dev.teelapp.io/api/payouts/PAYOUT_ID \
      -H "X-API-Key: $TEEL_API_KEY"
    ```

    For real-time updates, connect to the WebSocket endpoint:

    ```bash
    wscat -c "wss://dev.teelapp.io/ws" \
      -H "X-API-Key: $TEEL_API_KEY"
    ```

    The WebSocket pushes status changes as they happen: `pending`, `processing`, `completed`, or `failed`.

  </Step>
</Steps>
